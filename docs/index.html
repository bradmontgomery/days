Command Reference

Complete list of available options

usage: Print a list of days over a given number of weeks. [-h]
    [--start-date START_DATE]
    [-y YEAR] [-m MONTH]
    [-d DAY] [-n WEEKS]
    [-w [WEEKDAY ...]]
    [-g]

options:
     -h, --help show this help message and exit
     --start-date START_DATE
     Start date in YYYY-MM-DD format (overrides -y, -m, -d if provided)
     -y, --year YEAR Starting year (default: 2025)
     -m, --month MONTH Starting month (default: 12)
     -d, --dom DAY Day of month to start (default: 17)
     -n, --num-weeks WEEKS
     Number of weeks to print (default: 14)
     -w, --weekdays, --on [WEEKDAY ...]
     Weekdays to print (e.g., Tue Thu) (default: Tue Thu)
     -g, --group Group output by weeks
